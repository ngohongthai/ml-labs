## Khoa hc d liu

Vi Colab, bn c th khai thc ton b sc mnh ca cc th vin Python ph bin phn tch v trc quan ha d liu. cha m bn di s dng numpy to mt s d liu ngu nhin v s dng matplotlib trc quan ha d liu . chnh sa m ny, bn ch cn nhp vo v bt u chnh sa.

In [ ]:
import numpy as np
from matplotlib import pyplot as plt

ys = 200 + np.random.randn(100)
x = [x for x in range(len(ys))]

plt.plot(x, ys, '-')
plt.fill_between(x, ys, 195, where=(ys > 195), facecolor='g', alpha=0.6)

plt.title("Sample Visualization")
plt.show()
